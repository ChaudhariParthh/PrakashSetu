# 🌉 PrakashSetu

> An IoT-Based Smart Street Light Monitoring & Fault Detection System

PrakashSetu is a smart street lighting solution developed to enhance energy efficiency, automate street light operations, and simplify fault monitoring through IoT technologies. The project combines embedded systems, web technologies, and real-time communication to provide a modern approach to street light management.

The system is designed to monitor street lights, automate their operation based on environmental conditions, and provide timely notifications whenever a fault is detected. By integrating hardware and software components, PrakashSetu aims to contribute towards smarter and more sustainable urban infrastructure.

## ⚙️ Project Workflow

1. Environmental conditions are continuously monitored using sensors.
2. Street lights are automatically controlled based on the detected conditions.
3. The system monitors the operational status of connected street lights.
4. Fault conditions are identified and processed by the controller.
5. Fault information is transmitted through the IoT communication module.
6. Real-time notifications are delivered to the administrator.
7. The dashboard provides centralized monitoring and management capabilities.
